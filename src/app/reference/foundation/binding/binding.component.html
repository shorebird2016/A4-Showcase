What are the various Data Binding mechanisms?> Live Sample

One Way Class Property => DOM Element(Template) interpolation, [src]="imagePath"
Special notation bind-src is the same as [src]

One Way DOM Events => Class Property (keydown)

Attrib Binding Class Property => DOM Element Attributes [attrib.colspan]="15"

Class Binding Class Property => DOM Element CSS classes [class]="classForElem"

Style Binding Class Property => DOM Element CSS styles [ngStyle]="addStyles()"

Two Way Class Property <=> DOM Element(Template) [(ngModel)]
