How to write Structure Custom Directive ?
  1. Generate directive using cli, add to xxmodules.ts
  2. Import classes - TemplateRef, ViewContainerRef
  3. Add selector like [sbDir1] to @Directive decorator
  4. Add selector to HTML tag as a new attribute
  5. DI ViewContainerRef/TemplateRef to CTOR
  6. Invoke methods on thise classes to manipulate DOM structure
  7. Use @Input for additional parameters

How to write Attrib Custom Directive ?
  1. Generate directive using cli, add to xxmodules.ts
  2. Import classes - ElementRef, HostListener, Renderer2
  3. Add selector like [sbDir2] to @Directive decorator
  4. Add selector to HTML tag as a new attribute
  5. DI ElementRef/HostListener/Renderer2 to CTOR
  6. Invoke methods on thise classes to manipulate DOM properties (eg. ElementRef.nativeElement.style)
  7. Use @Input for additional parameters
