HTML View (1) bind [formGroup]="form" to class variable of type FormGroup

(3) bind events like (ngModelChange)="decimalChanged($event)"

Component Class (1) create property "form" from type FormGroup
(2) add member to this object for each field using "formControlName" from view
(3) create event handler "decimalChanged" for user changes
